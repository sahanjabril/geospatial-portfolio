
# Geospatial & Remote Sensing Portfolio: AI-Ready Spatial Data
This portfolio showcases advanced remote sensing and geospatial workflows optimized for machine learning integration, automated data pipelines, and large-scale mapping.

### Core Capabilities
* **AI Training Data Prep**: Generating high-quality, topologically clean vector training sets from satellite and aerial imagery.
* **PyQGIS Automation**: Developing custom Python scripts and QGIS Graphical Models to automate repetitive processing, clipping, and attribute assignment.
* **Large-Scale Mapping**: Executing automated raster analysis, land cover classification, and vector feature extraction.

### Featured Projects
1. Automated Feature Extraction Pipeline (PyQGIS)
   * Built a custom Python script within QGIS to automate the pre-processing, tiling, and vectorization of high-resolution imagery.
   * Cleansed topology defects automatically to ensure 100% error-free inputs for computer vision models.

2. Satellite Imagery Land Cover Classification
   * Processed multi-spectral satellite imagery using advanced remote sensing indices (NDVI, NDWI) to map target features.
   * Exported structured spatial data formats optimized for seamless AI ingestion.
